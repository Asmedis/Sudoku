# Optimized Sudoku solver
This solver can solve any size sudoku 3x3, 4x4, ...,  nxm

# Usage 
## Sudoku files
First you need to write a Sudoku file

The file is the whole sudoku writen out with spaces inbeetween number and \n for rows for empty tiles write 0

Set the file in the Sudoku.cpp file

# Running the program
For best performance compile the project with included run.bat

```run.bat```

To start the solver

```Sudoku.exe```

To compile without optimisation 

```g++ Sudoku.cpp -o Sudoku.exe```
